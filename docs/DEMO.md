## Beats

1. **Problem (15s)** — "Hypercasual studios burn money testing prototypes blind. A human has to play every prototype to know if it's fun — and most die anyway."
2. **Agent plays (25s — THE WOW)** — pre-recorded, sped-up clip of the agent playing the Mob Control clone through Computer Use + the report it produced. Punchline: *"It evaluates game feel through play, not instrumentation — no privileged access to game state."*
3. **Variants + videos (15s)** — show generated variants and Veo creatives.
4. **Dashboard + decision (15s — LIVE)** — live run view: event feed, metrics, keep/kill, next-build recommendation.
